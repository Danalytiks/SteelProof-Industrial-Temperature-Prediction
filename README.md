# SteelProof – Industrial Temperature Prediction


## Project Overview

SteelProof is a machine learning project focused on predicting the final temperature of molten steel during the production process. Accurate temperature prediction helps optimize energy consumption and improve efficiency in industrial steel manufacturing.

This project was developed as part of the **TripleTen Data Science Bootcamp**.

---

## Business Problem

Steel production requires large amounts of energy to heat metal to the required temperature. Without accurate temperature prediction, companies may waste energy or produce steel outside optimal conditions.

The goal of this project is to build a machine learning model capable of predicting the **final temperature of molten steel** based on operational process data.

---

## Data Analysis

The project includes an exploratory data analysis (EDA) to understand the dataset, including:

- dataset structure and size
- missing values and data quality
- statistical summaries of variables
- identification of the **target variable** and potential predictors

---

## Machine Learning Models

Several machine learning algorithms were tested to solve the regression problem:

- Linear Regression
- Random Forest
- Gradient Boosting

Models were evaluated and optimized using cross-validation and hyperparameter tuning.

---

## Evaluation Metrics

Model performance was evaluated using regression metrics:

- MAE (Mean Absolute Error)
- RMSE (Root Mean Squared Error)

These metrics help measure the prediction accuracy of the models.

The final model achieved strong predictive performance and demonstrates how machine learning can support **data-driven decision making in industrial environments**, helping optimize energy consumption and production efficiency.
